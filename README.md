# p2c_mech

Автоматизация P2C-агента и Telegram-бота для управления заявками.

## Стек

- Python 3.12
- FastAPI
- Aiogram 3
- PostgreSQL
- Redis
- Docker Compose

## Быстрый старт

```bash
docker compose up -d --build
```
